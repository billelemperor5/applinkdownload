# موقع تطبيق مصاريفي

هذا المجلد يحتوي على الموقع التعريفي للتطبيق.

## المحتويات
- `index.html`: الصفحة الرئيسية للموقع.
- `style.css`: ملف التنسيقات والتصميم.

## كيفية التعديل
1. **روابط التواصل الاجتماعي**:
   - افتح ملف `index.html`.
   - ابحث عن قسم `social-links`.
   - استبدل `#` في `href="#"` بروابط حساباتك الحقيقية (فيسبوك، تويتر، إلخ).

2. **رابط التحميل**:
   - رابط التحميل الحالي هو: `https://median.co/share/pwwweew#apk`.
   - إذا تغير الرابط مستقبلاً، يمكنك تعديله في قسم `hero` داخل `index.html`.

## النشر
يمكنك رفع محتويات هذا المجلد (website) إلى أي استضافة مجانية أو مدفوعة مثل:
- GitHub Pages
- Netlify
- Vercel
- استضافة خاصة بك
